README.TXT for ISMKEY.DBL

Description of function
-----------------------

This function will generate the parameters required to create an ISAM
file using the Synergy/DE Repository.  Any possible errors generated by
this routine are defined in ISMKEY.DEF

A standalone program can be generated by using the compiler directive
.define EXE_REQUIRED.

Uses the logical INC: to point to the include file directories.

Uses UTILITIES.ZIP
